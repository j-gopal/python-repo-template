
- .cache is for files created during development that would otherwise clutter the top-level repo dir
  - e.g. .pytest_cache, .mypy_cache
- .local is for storing data created while running the app on the user's PC
- To create a new project with an actual app name:
  $ cd path/to/python_repo_template 
  $ create_new_project.bat <actual_app_name> path/to/new_project
